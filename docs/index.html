üîß Installation

Step 1: Clone the Repository

git clone https://github.com/extremerom/tik-spyder.git
cd tik-spyder

Step 2: Install Dependencies

pip install -r requirements.txt

Or install as a package:

pip install -e .

Step 3: Configure API Keys

Edit config/config.ini with your API keys:

[SerpAPI Key]
api_key = your_serp_api_key

[Apify Token]
apify_token = your_apify_token

Step 4: Launch TikSpyder

# Web interface
tikspyder --app

# Command line
tikspyder --q "search term" --download

üìù Usage Examples

Search-based Collection

tikspyder --q "F-16 AND Enemy AND (Ukraine OR Russia)" \
  --gl us --hl en \
  --after 2024-02-01 --before 2024-05-31 \
  --download

Profile-based Collection

tikspyder --q Trump --user username \
  --apify --oldest-post-date 2025-01-01 \
  --download

Hashtag-based Collection

tikspyder --tag sinaloa \
  --apify --oldest-post-date 2025-08-01 \
  --number-of-results 50 \
  --download
